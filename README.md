# Food Delivery SQL Analytics

SQL analysis project for a food delivery platform using PostgreSQL.

## Topics Covered
- Revenue & profit analysis
- Monthly Active Users (MAU)
- Retention & growth metrics
- ARPU calculation
- User segmentation
- Window functions & CTEs
- Crosstab reporting

## Tech Stack
- PostgreSQL
- SQL
- CTEs
- tablefunc extension

## Dataset Tables
- `meals`
- `orders`
- `stock`
Dataset download links are included in the SQL script.

## Purpose
Practice real-world business analytics queries and build a SQL portfolio project.
